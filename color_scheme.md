# Dytron Infrastructure Website Color Scheme

| Element | Hex Code | Purpose | Visual Feel |
| :--- | :--- | :--- | :--- |
| **Canvas / Background** | `#0D0F12` | Main page background (Deep Obsidian) | Ultra-modern, premium dark-mode feel. |
| **Primary Accent** | `#0A2E5C` | Left pillar / Heavy structural elements | Industrial Deep Navy. Represents engineering and concrete stability. |
| **Secondary Accent** | `#E6951B` | Right pillar / Highlights / CTAs | Burnished Industrial Amber. Adds sharp contrast and energy. |
| **Card / Border** | `#1A1D24` | Component containers & thin borders | Subtle separation that keeps layouts crisp. |
| **Text Primary** | `#FFFFFF` | Headings & Hero text | High-impact readability. |
| **Text Secondary** | `#8A94A6` | Paragraphs & Technical specs | Muted slate gray to prevent eye strain. |
